# tradestation-python
TradeStation Python Client SDK. Note, you are expected to have received a `client id` and a `client secret` after funding an account. Please refer to TradeStation's [docs](https://api.tradestation.com/docs/faq) for more details.

## Install

Install with conda.

```bash
conda install jkim-mlops::tradestation-python
```

## Usage

```
from tradestation_python import TradeStation
```

For more details, please refer to the [example](./example/).
